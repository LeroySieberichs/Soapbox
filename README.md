
# SoapBox ICT opdracht

## Demo

 ![Application Demo](https://i.imgur.com/3hKnkUr.gif)
 
## Run Locally

Clone the project

```bash
  git clone https://github.com/LeroySieberichs/Soapbox/
```

Go to the project directory

```bash
  cd Soapbox
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
